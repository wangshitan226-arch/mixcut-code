# 开拍式网感剪辑功能接入方案

## 一、功能概述

在混剪结果生成后，为用户提供类似"开拍"App的后期处理能力，包括文字快剪、网感模板、字幕样式、BGM替换等功能。

---

## 二、用户交互流程

### 2.1 入口
用户在混剪结果页，点击某个视频的【网感剪辑】按钮，进入网感剪辑界面。

### 2.2 主界面布局
- **顶部**：返回按钮、标题、保存按钮
- **中间**：视频预览区（9:16竖屏比例）
- **底部**：功能Tab栏 + 当前功能面板

### 2.3 四大功能模块

#### 1) 文字快剪
**进入时**：系统正在识别视频语音，显示加载动画。识别完成后，展示每句话的文字及对应的时间位置。

**文字展示形式**：按时间顺序列出所有文字，语气词和静音段自动标记出来。

**用户操作**：
- 点击某段文字，视频跳转到该位置播放
- 选中文字后，可以删除该段（删除后该段文字变灰，对应视频片段将被剪掉）
- 点击【一键去水词】，自动选中所有"嗯、啊、这个、那个"等语气词
- 点击【删除气声】或【删除静音】，系统自动识别并标记对应片段
- 可以撤销和重做操作

**确认**：删除的片段会汇总显示，告知用户共删除了几段、节省了多少秒。

#### 2) 网感模板
**功能**：一键套用完整的视频包装模板。

**用户操作**：
- 选择模板分类（热门、带货、知识、Vlog、美食等）
- 浏览模板预览图，点击可预览效果
- 选择某个模板后，该模板包含的字幕样式、贴纸、转场、BGM自动应用到视频
- 可以在此基础上继续调整其他功能

#### 3) 字幕
**功能**：给视频添加字幕样式。

**用户操作**：
- 选择字幕模板（经典、综艺、电影等风格）
- 调节字体大小、颜色、描边粗细
- 选择字幕位置（底部、中部、顶部）
- 选择入场动画（弹出、渐显、打字机等）
- 点击【重新生成字幕】，系统根据视频语音重新生成字幕
- 点击【编辑字幕内容】，可以手动修改字幕文字

#### 4) BGM
**功能**：给视频添加背景音乐。

**用户操作**：
- 选择音乐分类（热门、轻快、励志、情感等）
- 在列表中选择某首音乐试听
- 调节原声和BGM的音量比例
- 选择是否开启淡入淡出效果

### 2.4 保存与导出

**点击保存**：
- 显示处理中状态，系统将视频进行云端渲染
- 渲染完成后，返回混剪结果页
- 原视频保留，新增一个"已剪辑"的版本显示在结果列表中
- 新版本带有【网感剪辑】的标记

**版本管理**：
- 一个原始视频可以有多个剪辑版本
- 用户可以对任意版本再次进行网感剪辑
- 所有版本都可以单独下载

## 三、开放接口接入点

### 3.1 语音识别接口

**接入时机**：用户进入网感剪辑界面时

**功能**：将视频语音转换为文字，并返回每个字的时间戳

**可选服务商**：
| 服务商 | 接口 | 特点 |
|--------|------|------|
| 阿里云 | FunASR / 智能语音交互 | 中文字识别准确率高，支持字级时间戳 |
| OpenAI | Whisper API | 词级时间戳精准，支持中文 |
| 科大讯飞 | 语音听写API | 中文识别率高，支持方言 |

**传入**：视频文件URL
**返回**：字级时间戳列表，包含每个字的文字内容、开始时间、结束时间、类型标记

---

### 3.2 静音/气声检测接口

**接入时机**：语音识别完成后，或用户点击【删除静音】【删除气声】按钮时

**功能**：自动识别视频中的静音段和语气词

**可选方案**：
| 方案 | 说明 |
|------|------|
| FunASR内置VAD | 阿里云语音识别自带语音活动检测 |
| Silero VAD | 开源VAD模型，可本地部署或调用API |
| 自研规则 | 基于语音识别结果，通过规则匹配语气词 |

**传入**：音频数据或语音识别结果
**返回**：静音段时间范围、语气词时间范围

---

### 3.3 模板列表接口

**接入时机**：用户切换到【网感模板】Tab时

**功能**：获取可用的视频模板列表

**可选方案**：
| 方案 | 说明 |
|------|------|
| 自建模板库 | 自行设计模板，存储在OSS/CDN |
| 剪映CapCut API | 调用剪映模板接口 |
| 七牛云/腾讯云 | 使用云服务商的模板市场 |

**传入**：模板分类
**返回**：模板列表，包含模板ID、名称、预览图URL、效果参数

---

### 3.4 视频渲染接口

**接入时机**：用户点击【保存】按钮时

**功能**：根据用户的剪辑操作，生成新的视频文件

**可选服务商**：
| 服务商 | 接口 | 特点 |
|--------|------|------|
| 七牛云 | 持久化处理(pfop) | 国内稳定，按量计费，支持视频裁剪拼接 |
| 腾讯云 | 云点播-云剪辑 | 功能完善，支持复杂剪辑操作 |
| 火山引擎 | 视频剪辑SDK | 字节跳动旗下，支持云端剪辑 |
| FFmpeg API | 自建/开源 | 灵活可控，需要自行部署 |

**传入**：
- 原视频URL
- 保留的时间段列表（文字快剪后）
- 字幕样式参数
- BGM音乐ID和音量设置
- 模板ID和效果参数

**返回**：新视频的URL、处理状态

---

### 3.5 字幕渲染接口

**接入时机**：用户调整字幕样式时实时预览，或最终保存时

**功能**：将字幕以指定样式叠加到视频上

**可选方案**：
| 方案 | 说明 |
|------|------|
| 火山引擎视频剪辑SDK | 支持实时预览和云端渲染 |
| 腾讯云点播字幕叠加 | 支持多种字幕样式 |
| FFmpeg burn字幕 | 使用ass/srt字幕文件叠加 |

---

### 3.6 BGM音乐库接口

**接入时机**：用户切换到【BGM】Tab时

**功能**：获取可用的背景音乐列表

**可选方案**：
| 方案 | 说明 |
|------|------|
| 自建音乐库 | 购买版权音乐，存储在OSS |
| 腾讯云音乐 | 使用云服务商提供的音乐库 |
| 免版权音乐 | 使用YouTube Audio Library等免费音乐 |

## 四、数据结构设计

### 4.1 字级时间戳
```typescript
interface WordTimestamp {
  id: string;
  word: string;
  start: number;        // 开始时间（秒）
  end: number;          // 结束时间（秒）
  type: 'normal' | 'silence' | 'filler' | 'punctuation';
  confidence: number;   // 识别置信度
  selected?: boolean;   // 是否被选中删除
}
```

### 4.2 剪辑版本
```typescript
interface EditedVersion {
  id: string;
  parent_id: string;    // 原视频ID
  version: number;
  preview_url: string;
  thumbnail: string;
  edit_params: {
    removed_segments: Array<{
      start: number;
      end: number;
      type: 'manual' | 'filler' | 'silence';
    }>;
    subtitle_style: {
      template: string;
      font_size: number;
      color: string;
      stroke: boolean;
      position: 'top' | 'center' | 'bottom';
      animation: string;
    };
    bgm: {
      music_id: string;
      volume_ratio: number;
      fade_in_out: boolean;
    };
    template: {
      template_id: string;
    };
  };
  created_at: string;
}
```

## 五、接口调用流程

```
用户点击【网感剪辑】
    ↓
调用语音识别API（阿里云FunASR / OpenAI Whisper / 讯飞）
    ↓
返回字级时间戳，展示文字快剪界面
    ↓
用户操作：
  - 删除片段（选中文字 → 点击删除）
  - 选择模板（切换Tab → 选择模板）
  - 调整字幕（切换Tab → 调整样式）
  - 更换BGM（切换Tab → 选择音乐）
    ↓
点击【保存】
    ↓
调用视频渲染API（七牛云pfop / 腾讯云点播 / FFmpeg）
    ↓
轮询查询处理进度
    ↓
处理完成，返回新视频URL
    ↓
返回混剪结果页，展示新版本视频
```

## 六、成本估算

以处理1小时视频为例：

| 环节 | 服务商 | 费用 |
|------|--------|------|
| 语音识别 | 阿里云FunASR | ¥3-8/小时 |
| 静音检测 | FunASR内置VAD | 免费 |
| 视频渲染 | 七牛云pfop | ¥0.1-0.5/次 |
| **总计** | | **约 ¥5-10/小时视频** |

---

## 七、推荐接入方案

### 国内用户（最优性价比）
1. **语音识别**：阿里云 FunASR（中文准确率高，支持字级时间戳）
2. **视频渲染**：七牛云持久化处理pfop（稳定，按量计费）
3. **模板/BGM**：自建（购买商用版权）

### 海外用户（最简单）
1. **语音识别**：OpenAI Whisper API（API简单易用）
2. **视频渲染**：Cloudflare Stream / 自建FFmpeg
3. **模板/BGM**：自建
