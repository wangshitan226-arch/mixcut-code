# 阿里云ICE字幕模板参数完整文档

## 一、基础参数（SubtitleTrackClip）

### 1.1 必需参数

| 参数名 | 类型 | 说明 | 示例 |
|-------|------|------|------|
| Type | String | 素材类型，固定为"Text" | "Text" |
| Content | String | 字幕内容，支持\n换行 | "主标题\n副标题" |
| TimelineIn | Float | 在时间线上的开始时间（秒） | 0.0 |
| TimelineOut | Float | 在时间线上的结束时间（秒） | 5.0 |

### 1.2 位置参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| X | Float | X坐标位置 | 0~1（相对比例）或像素值 | 0.5（居中） |
| Y | Float | Y坐标位置 | 0~1（相对比例）或像素值 | 0.35（上方） |
| Alignment | String | 对齐方式 | "Center", "Left", "Right", "TopCenter", "BottomCenter" | "Center" |

### 1.3 字体样式参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| Font | String | 字体名称 | 系统字体或自定义字体 | "AlibabaPuHuiTi-Heavy" |
| FontURL | String | 自定义字体OSS地址 | OSS URL | "https://xxx/font.ttf" |
| FontSize | Integer | 字体大小 | 4~120（像素） | 80 |
| FontColor | String | 字体颜色 | #RRGGBB格式 | "#FFD700"（金色） |
| FontColorOpacity | Float | 字体透明度 | 0~1 | 1.0（不透明） |
| FontFace | Object | 字体样式 | {Bold, Italic, Underline} | {"Bold": true} |

### 1.4 描边参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| Outline | Integer | 描边宽度 | 0~10（像素） | 4 |
| OutlineColour | String | 描边颜色 | #RRGGBB格式 | "#8B4513" |

### 1.5 阴影参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| Shadow | Integer | 阴影深度 | 0~10 | 3 |
| BackColour | String | 阴影/背景颜色 | #RRGGBB格式 | "#000000" |
| ShadowOffset | Float | 阴影偏移 | 像素值 | 2.0 |

### 1.6 背景框参数（Effects中配置）

| 参数名 | 类型 | 说明 | 示例 |
|-------|------|------|------|
| Type | String | 效果类型 | "Box" |
| Color | String | 背景颜色 | "000000" |
| Opacity | String | 背景透明度 | "0.4" |
| Radius | Float | 圆角半径 | 0.1 |

## 二、动画效果参数

### 2.1 入场动画参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| AaiMotionIn | Float | 入场动画时长 | 0~10（秒） | 0.8 |
| AaiMotionInEffect | String | 入场动画效果 | 见下方动画列表 | "rotateup_in" |

### 2.2 出场动画参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| AaiMotionOut | Float | 出场动画时长 | 0~10（秒） | 0.5 |
| AaiMotionOutEffect | String | 出场动画效果 | 见下方动画列表 | "close_out" |

### 2.3 循环动画参数

| 参数名 | 类型 | 说明 | 取值范围 | 示例 |
|-------|------|------|---------|------|
| AaiMotionLoop | Integer | 循环次数 | 0~10 | 2 |
| AaiMotionLoopEffect | String | 循环动画效果 | "bounce", "pulse", "shake" | "bounce" |

## 六、当前系统中的字幕模板配置

### 6.1 主标题（#）

```python
{
    "font": "AlibabaPuHuiTi-Heavy",
    "font_size": 90,
    "font_color": "#FFD700",      # 金色
    "outline": 5,                  # 粗描边
    "outline_color": "#8B4513",   # 棕色描边
    "motion_in": "rotateup_in",   # 螺旋入场
    "motion_out": "close_out",    # 收缩出场
    "y": 0.35,                     # 位置偏上
    "weight": 1.5                  # 时长权重
}
```

### 6.2 副标题（##）

```python
{
    "font": "AlibabaPuHuiTi-Regular",
    "font_size": 50,
    "font_color": "#FFFFFF",      # 白色
    "outline": 3,
    "outline_color": "#333333",
    "motion_in": "fade_in",
    "motion_out": "fade_out",
    "y": 0.52,
    "weight": 1.2
}
```

### 6.3 章节标题（###）

```python

    "font_size": 70,
    "font_color": "#00FFFF",      # 青色

    "motion_out": "wave_out",

    "weight": 1.2
}
```

### 6.4 重点强调（!）


## 八、使用建议

### 8.1 字号推荐

| 用途 | 推荐字号 | 说明 |
|-----|---------|------|
| 主标题 | 80-120 | 醒目大字 |
| 副标题 | 50-70 | 中等大小 |
| 正文 | 40-50 | 清晰可读 |
| 提示 | 45-55 | 次要信息 |

### 8.2 颜色推荐

| 用途 | 推荐颜色 | 色值 |
|-----|---------|------|
| 主标题 | 金色 | #FFD700 |
| 副标题 | 白色 | #FFFFFF |
| 强调 | 黄色 | #FFDD00 |
| 数据 | 绿色 | #00FF88 |
| 警告 | 红色 | #FF4444 |

### 8.3 位置推荐（Y坐标）

| 位置 | Y值 | 说明 |
|-----|-----|------|
| 顶部 | 0.15-0.25 | 章节标题 |
| 中上 | 0.30-0.40 | 主标题 |
| 中部 | 0.45-0.55 | 强调内容 |
| 中下 | 0.60-0.75 | 正文 |
| 底部 | 0.80-0.90 | 提示信息 |
